# Smart Attendance System Using Face Recognition
**Course:** CIT310 - Information Technology Project
**Project ID:** CIT310_01_26_77
**Project Type:** Software-Based / Both Software and Hardware

## 📝 Project Overview
The Smart Attendance System using Face Recognition automates attendance recording and improves efficiency in educational institutions. The system includes face detection, face recognition, student registration, automatic attendance recording, and report generation for classroom use.

## 🚩 Problem Statement
* Manual attendance systems are time consuming, prone to human error, and allow proxy attendance.
* An automated facial recognition system can improve accuracy and efficiency.

## 🎯 Project Objectives
* **Main Objective:** Develop an automated attendance management system using face recognition.
* Capture student faces using a webcam.
* Train a face recognition model.
* Automatically mark attendance.
* Store records in a database.
* Generate attendance reports.

## 🛠️ Required Resources
* **Programming Language:** Python
* **Libraries:** OpenCV, NumPy, Face Recognition
* **Database:** MySQL/SQLite
* **Development Tools:** Visual Studio Code
* **Hardware:** Computer and Webcam

## 📈 Methodology
The system follows the Software Development Life Cycle (SDLC):
1. **Requirement Analysis:** Identifying system needs and gathering constraints.
2. **System Design:** Creating the architectural and database schema.
3. **Development:** Programming the core logic and interface.
4. **Testing:** Verifying recognition accuracy and system stability.
5. **Deployment:** Final implementation and setup for use.
